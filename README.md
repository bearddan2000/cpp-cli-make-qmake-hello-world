# cpp-cli-make-qmake-hello-world

## Description
This is a POC project to demonstrate 
make with qmake as a build tool.

## Tech stack
- cpp
- make
- qt5
    - qmake

## Docker stack
- alpine:edge

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
